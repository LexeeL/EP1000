How to Fusion 360 (3D Printing)

Navigation:

This page is explaining the process of 3D Printing using the Fusion 360 program

Step 1
  • Draw a grid of 30mm x 50mm
    • This is to draw a grid to limit the size of how big the chess piece should be

    Step 2

  • Resize the canvas image/ reference image to fit the grid
    • By using th grid as a limiter, the size can be set accurately

    Step 3

  • Draw a horizontal line to seperate base and body
    • This is to help visualise which part you should dimension first

    Step 4

  • Trace the knight body using spline or line
    • Spline will help make the entire design smooth instead of using a line which might end up jagged

    Step 5

  • Trace the knight base and fillet it
    • Now to dimension the base and using the horizontal line, you can make sure both sketches are in contact of eachother

    Step 6

  • Extrude body to 30mm wide and revolve base
    • This is to give the 2D sketch it's shape

    Step 7

  • Use extrude or hole/different functions to add detail to the knight
    • This is up to your imagination on how detailed of a knight piece you would want to make
